# VisaPilot AI 🌍✈️

## Overview

VisaPilot AI is an AI-powered visa and immigration assistant designed to simplify the visa application journey for Indian citizens.

## Features

- Visa Eligibility Assessment
- AI Visa Recommendations
- Document Checklist Generator
- Application Readiness Score
- Application Tracking Dashboard
- AI Immigration Assistant

## Technology Stack

### Frontend
- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel

### Backend
- Node.js
- Next.js API Routes

### Database
- AWS Aurora PostgreSQL

### AI
- OpenAI API

## Vision

VisaPilot AI aims to become the world's most trusted AI-powered immigration copilot.
